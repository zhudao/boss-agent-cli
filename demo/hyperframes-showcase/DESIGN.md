# boss-agent-cli Showcase Visual Identity

## Style Prompt

A precise product-launch animation for an open-source command-line tool built for AI agents. The look should feel like a maintained engineering project, not a recruitment ad: dark terminal surfaces, warm paper-tinted text, strict grid composition, restrained motion, and small green/teal signals for successful machine-readable workflows.

## Colors

- Background: `#0e0c09`
- Surface: `#17130f`
- Ink: `#f2ede4`
- Muted ink: `#a7988b`
- Accent: `#008b8b`
- Success: `#47c772`
- Python blue: `#3776ab`
- Warning amber: `#d99a31`

## Typography

- Display: `Avenir Next`, `PingFang SC`, `Hiragino Sans GB`, system sans-serif
- Data/code: `JetBrains Mono`, `Menlo`, `Consolas`, monospace
- Editorial contrast: `Georgia`, `Songti SC`, serif

## Motion

- Primary transition: teal cover wipe with brief code-line shimmer.
- Secondary transition: soft terminal focus pull.
- Entrance timing: 0.35s to 0.75s with staggered command-line and schema elements.
- Ambient motion: slow grid drift, status pulse, and faint code stream movement.

## What NOT to Do

- Do not use generic blue-purple neon gradients.
- Do not turn the project into a marketing landing page.
- Do not show fake web dashboards as the primary product surface.
- Do not use cartoon recruiter/job-seeker illustrations.
- Do not animate command output in a way that violates the stdout JSON envelope concept.
